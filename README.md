# Ninth Inning

A single-file HTML baseball game. First-person batter's POV — read the pitch, time your swing, go for the fences.

Open `baseball.html` in any modern browser (desktop, iPad Safari, etc.). No build step, no dependencies.

## Controls

**Desktop**
- `Space` or click — swing
- `A` / `D` or `←` / `→` — aim contact point (inside/outside)
- `W` / `S` or `↑` / `↓` — aim height

**iPad / touch**
- Tap **SWING** button
- Drag the **AIM** pad to set contact point

## Features

- Five pitch types with distinct break and tempo: fastball, curveball, slider, changeup, sinker
- Outcomes: home run, triple, double, single, foul, fly out, ground out, line out, strikeout, walk
- Nine-inning game vs. CPU plus an infinite Practice mode
- Walk-off rule
- Synthesized crowd noise that swells on hits, organ fanfare on home runs, bat crack and whiff sounds — no audio files
- Mini diamond and scoreboard HUD
